BLOG IMAGES DIRECTORY
=====================

Place blog post featured images in this directory.

Recommended specifications:
- Format: JPG or WebP
- Dimensions: 1200x630px (optimal for social sharing)
- File size: Under 200KB for fast loading
- Naming convention: Use the blog post slug + .jpg
  Example: coachella-valley-market-report-2024-2025.jpg

Current blog posts expecting images:
1. market-report-2024.jpg
2. first-home-palm-desert.jpg

For best AIO results:
- Use descriptive alt text in the blog post data
- Include location-specific imagery when possible
- Optimize images before uploading (compress, resize)
