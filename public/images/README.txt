PLACEHOLDER IMAGES DIRECTORY
============================

This directory is for your website images. Please add:

1. hero-desert.jpg
   - Beautiful Coachella Valley landscape photo
   - Recommended size: 1920x800px
   - Use as homepage hero background
   - Should showcase desert mountains, golf courses, or palm trees

2. brenda-portrait.jpg (optional)
   - Professional headshot of Brenda Devlin
   - Recommended size: 400x400px
   - Can be used in About section

3. property-*.jpg (optional)
   - Featured property images
   - Showcase your best listings

Image Optimization Tips:
- Use WebP format for faster loading
- Compress images (aim for under 200KB for web)
- Use descriptive alt text for accessibility
- Consider using Next.js Image component for automatic optimization
